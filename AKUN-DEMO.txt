CATATAN KEAMANAN:
=================
⚠️ PENTING: Ganti password default setelah login pertama!

Untuk production:
1. Ganti semua password default
2. Hapus akun demo yang tidak diperlukan
3. Set APP_DEBUG=false di file .env
4. Gunakan password yang kuat


CARA GANTI PASSWORD:
====================
1. Login sebagai superuser
2. Menu: Manajemen User
3. Klik Edit pada user yang ingin diganti password
4. Isi password baru
5. Klik Update


CARA TAMBAH USER BARU:
=======================
1. Login sebagai admin atau superuser
2. Menu: Manajemen User
3. Klik: Tambah User
4. Isi form lengkap
5. Klik: Simpan


WORKFLOW PENGGUNAAN:
====================

Sebagai SUPERUSER:
1. Cek departemen (sudah otomatis dibuat)
2. Buat admin untuk departemen yang belum ada
3. Monitor semua aktivitas

Sebagai ADMIN:
1. Buat shift kerja (pagi, siang, malam)
2. Buat akun user (magang/daily worker)
3. Buat jadwal kerja untuk user
4. Monitor absensi harian
5. Export laporan berkala

Sebagai USER:
1. Login setiap hari
2. Cek jadwal shift
3. Check-in saat mulai kerja
4. Check-out saat selesai kerja
5. Lihat riwayat absensi
