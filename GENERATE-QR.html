Hotel Attendance System

📱 Generate QR Code for Easy Access
Users can scan this QR code to quickly access the attendance system from their phones.

📋 How to Use:

  1. Run CHECK-IP.bat to get your IP address
  2. Enter the IP address above (e.g., 192.168.1.100)
  3. Click "Generate QR Code"
  4. Print or display the QR code
  5. Users scan with their phone camera
  6. They will be directed to the login page

Note: Make sure the server is running with SERVE-LAN.bat
