📌 FILE UTAMA (GUNAKAN INI):
═════════════════════════════

🔵 README-FIRST.txt
   → BACA INI DULU! Panduan lengkap untuk memulai

🔵 START-HERE.bat
   → Menu utama untuk mengakses semua fungsi
   → Gunakan ini untuk operasional sehari-hari

🔵 SETUP-LENGKAP.bat
   → Setup otomatis dari awal sampai selesai
   → Untuk instalasi pertama kali


📂 FILE INSTALASI:
═══════════════════

1️⃣ 1-install.bat
   Fungsi: Install composer dependencies
   Kapan: Pertama kali setup atau setelah update code
   
2️⃣ 2-migrate.bat
   Fungsi: Migrasi database dan seed data demo
   Kapan: Setelah buat database atau reset database


🚀 FILE OPERASIONAL:
═════════════════════

3️⃣ 3-serve.bat
   Fungsi: Jalankan development server
   Kapan: Setiap kali ingin menggunakan sistem
   URL: http://localhost:8000

4️⃣ 4-reset-database.bat
   Fungsi: Reset database (hapus semua data)
   Kapan: Ingin mulai dari awal atau testing
   ⚠️ HATI-HATI: Akan menghapus semua data!


📱 FILE WHATSAPP:
══════════════════

5️⃣ 5-send-whatsapp-report.bat
   Fungsi: Kirim laporan absensi via WhatsApp (manual)
   Kapan: Ingin kirim laporan kapan saja
   Syarat: WhatsApp API sudah dikonfigurasi

6️⃣ 6-run-scheduler.bat
   Fungsi: Jalankan scheduler untuk kirim laporan otomatis
   Kapan: Ingin laporan terkirim otomatis setiap hari
   Jadwal: Setiap hari jam 17:00


🔧 FILE MAINTENANCE:
═════════════════════

7️⃣ 7-clear-cache.bat
   Fungsi: Bersihkan cache aplikasi
   Kapan: Saat development atau ada masalah cache

8️⃣ 8-optimize.bat
   Fungsi: Optimize aplikasi untuk production
   Kapan: Sebelum deploy ke production


📖 FILE DOKUMENTASI:
═════════════════════

📄 BUKA-DOKUMENTASI.bat
   Fungsi: Buka semua file dokumentasi
   Isi: README, Cara Menjalankan, Akun Demo, dll


═══════════════════════════════════════════════════════════


🎯 WORKFLOW PENGGUNAAN:
════════════════════════

INSTALASI PERTAMA KALI:
------------------------
1. Pastikan Laragon running
2. Jalankan: SETUP-LENGKAP.bat
3. Database akan dibuat otomatis
4. Login dengan akun demo


PENGGUNAAN SEHARI-HARI:
------------------------
1. Jalankan: START-HERE.bat
2. Pilih menu [3] Jalankan Server
3. Buka browser: http://localhost:8000
4. Login dan gunakan sistem


KIRIM LAPORAN WHATSAPP:
------------------------
1. Konfigurasi WhatsApp API di .env
2. Jalankan: 5-send-whatsapp-report.bat
3. Atau setup otomatis: 6-run-scheduler.bat


RESET SISTEM:
-------------
1. Jalankan: 4-reset-database.bat
2. Semua data akan dihapus
3. Data demo akan dibuat ulang
