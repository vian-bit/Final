👨‍💼 PANDUAN UNTUK ADMIN:
==========================

CARA MEMBUAT JADWAL SEBULAN:
-----------------------------

1. Login sebagai admin:
   Email: admin.fo@hotel.com
   Password: password

2. Klik menu "📅 Kelola Jadwal"
   Atau: http://localhost:8000/schedules-calendar

3. Pilih User:
   - Dropdown akan menampilkan semua user di departemen Anda
   - Pilih user yang ingin diatur jadwalnya

4. Atur Jadwal:
   - Setiap tanggal ada dropdown shift
   - Pilih shift untuk hari tersebut
   - Atau pilih "Libur" jika tidak kerja


TOMBOL ATUR CEPAT:
------------------

1. "Semua Shift Pagi"
   → Atur semua hari dengan Shift Pagi

2. "Semua Shift Siang"
   → Atur semua hari dengan Shift Siang

3. "Semua Shift Malam"
   → Atur semua hari dengan Shift Malam

4. "Senin-Jumat"
   → Atur hanya hari kerja (Senin-Jumat)
   → Akan minta input ID shift

5. "Hapus Semua"
   → Hapus semua jadwal (set ke Libur)


CARA MENGGUNAKAN TOMBOL CEPAT:
-------------------------------

Contoh 1 - Semua Shift Pagi:
1. Klik "Semua Shift Pagi"
2. Konfirmasi
3. Semua tanggal akan diset Shift Pagi

Contoh 2 - Senin-Jumat Saja:
1. Klik "Senin-Jumat"
2. Masukkan ID shift (lihat di dropdown)
3. Hanya Senin-Jumat yang diset
4. Sabtu-Minggu tetap Libur


CARA MENYIMPAN:
---------------
1. Setelah selesai atur jadwal
2. Klik tombol "💾 Simpan Semua Jadwal"
3. Tunggu konfirmasi berhasil
4. Jadwal akan tersimpan


TIPS EFISIEN:
-------------

Skenario 1 - Shift Tetap:
User kerja Shift Pagi setiap hari
→ Klik "Semua Shift Pagi"
→ Simpan

Skenario 2 - Senin-Jumat Saja:
User kerja Senin-Jumat, libur weekend
→ Klik "Senin-Jumat"
→ Input ID shift
→ Simpan

Skenario 3 - Custom:
User kerja shift berbeda-beda
→ Atur manual per tanggal
→ Simpan

Skenario 4 - Edit Beberapa Hari:
→ Atur cepat dulu (misal: Semua Shift Pagi)
→ Edit manual tanggal tertentu yang berbeda
→ Simpan

📅 WORKFLOW LENGKAP:
====================

AWAL BULAN (Admin):
-------------------
1. Login sebagai admin
2. Buka "Kelola Jadwal"
3. Pilih bulan depan
4. Untuk setiap user:
   a. Pilih user dari dropdown
   b. Atur jadwal sebulan
   c. Gunakan tombol cepat untuk efisiensi
   d. Simpan

TENGAH BULAN (Admin):
---------------------
1. Jika ada perubahan jadwal
2. Buka "Kelola Jadwal"
3. Pilih user
4. Edit tanggal yang perlu diubah
5. Simpan


SETIAP HARI (User):
-------------------
1. Login
2. Lihat "Jadwal Saya"
3. Cek jadwal hari ini
4. Pergi ke dashboard
5. Check-in sesuai jadwal

💡 TIPS & TRIK:
===============

UNTUK ADMIN:
------------

1. Atur Jadwal di Awal Bulan:
   - Lebih mudah daripada atur per hari
   - User bisa lihat jadwal lebih awal
   - Lebih terorganisir

2. Gunakan Tombol Cepat:
   - Hemat waktu
   - Konsisten
   - Mudah diubah jika perlu

3. Cek Ulang Sebelum Simpan:
   - Scroll kalender
   - Pastikan tidak ada yang terlewat
   - Perhatikan hari libur nasional

4. Koordinasi dengan User:
   - Tanya preferensi shift
   - Perhatikan request libur
   - Komunikasi perubahan jadwal


UNTUK USER:
-----------

1. Cek Jadwal Rutin:
   - Setiap awal minggu
   - Sebelum tidur untuk besok
   - Saat ada notifikasi perubahan

2. Screenshot Jadwal:
   - Untuk backup
   - Jika tidak bisa akses sistem
   - Share ke keluarga

3. Lapor Jika Ada Masalah:
   - Jadwal tidak sesuai
   - Tidak bisa check-in
   - Shift berubah mendadak

🔍 TROUBLESHOOTING:
===================

Jadwal tidak muncul?
--------------------
- Refresh browser (F5)
- Clear cache
- Cek bulan yang dipilih
- Pastikan admin sudah buat jadwal


Tidak bisa simpan jadwal?
--------------------------
- Cek koneksi internet
- Pastikan semua field terisi
- Coba refresh dan isi ulang
- Cek log error


Tombol cepat tidak jalan?
--------------------------
- Refresh browser
- Clear cache browser
- Coba browser lain
- Cek JavaScript enabled


User tidak muncul di dropdown?
-------------------------------
- Pastikan user aktif (is_active = true)
- Pastikan user role = 'user'
- Pastikan user di departemen yang sama
- Refresh halaman

🎯 BEST PRACTICES:
==================

1. Konsistensi:
   - Gunakan pola shift yang sama
   - Jangan sering ubah jadwal
   - Beri tahu jauh-jauh hari jika ada perubahan

2. Komunikasi:
   - Tanya preferensi user
   - Beri tahu perubahan
   - Dengar feedback

3. Fleksibilitas:
   - Akomodasi request libur
   - Pertimbangkan kondisi user
   - Buat backup plan

4. Dokumentasi:
   - Simpan jadwal lama
   - Catat perubahan
   - Arsip untuk referensi
