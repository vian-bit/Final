🎯 HOW TO USE ON MOBILE:
=========================

STEP 1 - Access from Phone:
----------------------------
1. Connect to same WiFi as server
2. Open browser (Chrome/Safari)
3. Type: http://192.168.1.100:8000
4. Bookmark for easy access


STEP 2 - Add to Home Screen:
-----------------------------
iPhone/iPad:
1. Tap Share button
2. Tap "Add to Home Screen"
3. Name it "Attendance"
4. Tap "Add"
5. Icon appears on home screen

Android:
1. Tap Menu (3 dots)
2. Tap "Add to Home screen"
3. Name it "Attendance"
4. Tap "Add"
5. Icon appears on home screen


STEP 3 - Use Like an App:
--------------------------
1. Tap the icon on home screen
2. Opens like a native app
3. No browser UI
4. Full screen experience

❓ TROUBLESHOOTING:
===================

Text Too Small?
---------------
- Pinch to zoom
- Browser will remember zoom level
- Or use accessibility settings


Buttons Hard to Tap?
--------------------
- Buttons are now larger
- Tap center of button
- Wait for visual feedback


Menu Won't Open?
----------------
- Refresh page (pull down)
- Clear browser cache
- Try different browser


Slow Loading?
-------------
- Check WiFi signal
- Move closer to router
- Restart browser
- Clear cache


Can't Add to Home Screen?
--------------------------
- Use Chrome or Safari
- Follow exact steps above
- Some browsers don't support
- Bookmark as alternative
